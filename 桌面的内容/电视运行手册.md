## 1. 准备工作

### 1.1 需要准备的东西

| 物品 | 用途 | 备注 |
|-----|------|------|

## 2. 方式一：电脑投屏（最简单）⭐推荐

**原理：** 电脑运行应用，通过投屏把画面传到电视，宝宝看大屏，家长在电脑端操作

```
电脑（运行应用+摄像头） → 投屏 → 电视（显示画面）
```

### 2.1 第一步：获取电脑IP地址

**Windows 电脑：**
1. 按 `Win + R` 打开运行框
2. 输入 `cmd`，回车
3. 在黑窗口输入：`ipconfig`
4. 找到「无线局域网适配器 WLAN」下的「IPv4 地址」
5. 记下来，格式类似：`192.168.1.100`

**Mac 电脑：**
1. 点击苹果菜单 → 系统设置
2. 点击「网络」
3. 点击「Wi-Fi」→「详细信息」
4. 找到「IP 地址」
5. 记下来

### 2.2 第二步：启动应用

1. 在电脑上打开浏览器，访问开发服务器地址（这个需要我帮你部署后给你）
2. 或者我们稍后一起完成这个步骤

### 2.3 第三步：Windows 电脑投屏到电视

**方法A：Win+K 快捷投屏（推荐）**

1. 确保电脑和电视连接同一个 WiFi
2. 在电脑上按 `Win + K`
3. 选择你的 TCL 雷鸟电视
4. 等待连接成功

**方法B：设置菜单投屏**

1. 电脑：设置 → 系统 → 投影到此电脑
2. 选择「所有网络都可投屏」
3. 电视：设置 → 常用设置 → 投屏 → 开启「镜像投屏」
4. 电脑开始投屏搜索，选择电视连接

### 2.4 第四步：电视端操作

1. 电视会自动显示电脑画面
2. 用鼠标在电视上操作应用（电视会同步显示）
3. 摄像头插到电脑上（不用插电视）

### 投屏方案优点
- ✅ 摄像头即插即用，无需额外设置
- ✅ 电脑端操作更方便（鼠标/触控板）
- ✅ 网络延迟低，体验流畅
- ✅ 无需复杂的电视端配置

### 投屏方案缺点
- ❌ 电视端只能显示，不能直接触摸操作
- ❌ 电脑需要一直开着

## 3. 方式二：直接部署到云端

**原理：** 把应用部署到网上，电视浏览器直接访问，无需电脑

### 3.1 第一步：安装 Node.js

1. 访问 https://nodejs.org/
2. 下载 LTS 版本（推荐 v18 或 v20）
3. 安装，一直点「下一步」即可

### 3.2 第二步：获取应用代码

1. 我会帮你生成完整的项目文件
2. 解压到电脑任意位置（如桌面）

### 3.3 第三步：安装依赖

1. 打开「命令提示符」（Win+R，输入 `cmd`）
2. 进入项目目录：
   ```bash
   cd Desktop/baby-talk
   ```
3. 安装依赖：
   ```bash
   npm install
   ```
4. 等待安装完成（约2-3分钟）

### 3.4 第四步：构建应用

```bash
npm run build
```

等待构建完成，会生成 `dist` 文件夹。

### 3.5 第五步：部署到 Vercel（免费）

1. 注册 Vercel 账号：https://vercel.com
2. 安装 Vercel CLI：
   ```bash
   npm install -g vercel
   ```
3. 在项目目录执行：
   ```bash
   vercel
   ```
4. 按提示操作：
   - 要部署到现有项目？→ 选择「N」（新建）
   - 项目名称？→ 输入 `yiyaxueyu`
   - 目录？→ 直接回车（当前目录）
5. 等待部署完成
6. 最后会显示访问地址，如：`https://yiyaxueyu.vercel.app`
7. **把这个地址记下来！**

### 3.6 第六步：电视浏览器打开应用

**TCL 雷鸟电视操作步骤：**

1. 打开电视，进入主界面
2. 找到「应用商店」或「我的应用」
3. 下载「UC浏览器」或「当贝市场浏览器」（推荐 UC）
4. 打开浏览器
5. 在地址栏输入你的 Vercel 地址（如 `https://yiyaxueyu.vercel.app`）
6. 按回车访问

### 3.7 部署方案优点
- ✅ 无需电脑一直开着
- ✅ 电视独立运行
- ✅ 可添加到电视桌面
- ✅ 离线也能用（已缓存的资源）

### 部署方案缺点
- ❌ 摄像头需要插到电视上
- ❌ 需要电视端配置摄像头权限

## 4. 摄像头连接与设置

### 4.1 摄像头连接（仅适用于云端部署方案）

**连接步骤：**

1. 将 USB 摄像头插入电视的 USB 接口
   - 推荐位置：电视顶部正中
   - 或电视底部某个 USB 口
   
2. 等待摄像头识别（约10秒）

3. 测试摄像头：
   - 打开电视的「相机」或「自拍」应用（部分电视有）
   - 确认能看到画面

### 4.2 电视浏览器摄像头权限

1. 打开 UC 浏览器
2. 访问应用地址
3. 浏览器会弹出提示：「是否允许使用摄像头」
4. 选择「允许」
5. 如果没有弹出：
   - 点击浏览器地址栏左侧的锁图标
   - 找到「摄像头」设置
   - 改为「允许」

### 4.3 摄像头摆放建议

```
         电视屏幕
    ┌─────────────────┐
    │                 │
    │      👶          │  ← 宝宝坐在电视前2-3米
    │                 │
    │    📷 ← 摄像头   │
    └─────────────────┘
    
摄像头建议：
- 位置：电视顶部或底部
- 角度：向下倾斜对准宝宝
- 距离：宝宝手臂能进入画面即可
```

### 4.4 推荐摄像头排行榜（适合电视使用）

| 排名 | 型号 | 价格 | 推荐理由 |
|-----|------|------|---------|
| 🥇 | 罗技 C920 | 280元 | 画质清晰，对焦快 |
| 🥈 | 罗技 C270 | 100元 | 性价比高，够用 |
| 🥉 | 奥尼 C10 | 80元 | 便宜，兼容性好 |

## 6. 日常使用流程

### 6.1 开机流程（推荐顺序）

```
1. 打开电视
   ↓
2. 插入USB摄像头（如果还没插）
   ↓
3. 打开UC浏览器
   ↓
4. 输入应用地址，按确认键
   ↓
5. 等待加载完成
   ↓
6. 可以开始使用啦！
```

### 6.2 使用场景

**场景A：动物乐园**
- 宝宝指向某个动物 → 动物卡片高亮
- 指向1.5秒 → 自动选中 → 播放动物叫声和发音
- 手势：握拳=暂停，张开=继续

**场景B：家庭成员**
- 宝宝说"妈妈" → 自动匹配妈妈卡片
- 宝宝说"爸爸" → 自动匹配爸爸卡片
- 显示鼓励语："真棒！宝宝认识家人了！"

**切换场景：**
- 手势：挥手
- 或点击底部切换按钮

### 6.3 关机流程

```
1. 按遥控器返回键，退出浏览器
2. （可选）拔掉USB摄像头
3. 关闭电视
```

## 7. 常见问题解答

### Q1：电视浏览器打不开应用？

**A：** 按以下顺序检查：
1. 电视和电脑/云端是否在同一网络？
2. 地址输入是否正确？（区分大小写）
3. 尝试刷新页面（按遥控器菜单键 → 刷新）
4. 换个浏览器试试（推荐UC浏览器）

### Q2：摄像头不被识别？

**A：** 排查步骤：
1. 摄像头插入电视USB口了吗？
2. 尝试其他USB口（有些口只供电不识别设备）
3. 换个摄像头测试
4. 电视是否支持该摄像头？（大部分UVC摄像头都支持）

### Q3：摄像头权限被拒绝？

**A：** 解决方法：
1. 电视设置 → 应用 → UC浏览器 → 权限 → 允许摄像头
2. 或者清除浏览器数据后重新访问

### Q4：手势识别不灵敏？

**A：** 优化建议：
1. 确保光线充足（背光会影响识别）
2. 手部动作要大一些
3. 手离摄像头近一些（1-3米最佳）
4. 确保手部在摄像头画面中央

### Q5：语音识别不准确？

**A：** 改善方法：
1. 确保麦克风权限已开启
2. 宝宝说话要靠近电视（1-2米内）
3. 环境要相对安静
4. 发音尽量标准（但系统也支持模糊识别）

### Q6：应用加载很慢？

**A：** 解决方案：
1. 首次加载慢是正常的（需要缓存资源）
2. 等待2-3分钟后再试
3. 检查网络速度
4. 使用离线模式（第二次加载会更快）

### Q7：画面卡顿？

**A：** 处理方法：
1. 关闭其他占用网络的程序
2. 重启路由器
3. 电视和路由器放近一些

## 8. 家长设置说明

### 8.1 如何打开家长设置

**方法：** 连续按5次遥控器「返回」键

### 8.2 设置选项

| 设置项 | 功能 | 推荐 |
|-------|------|------|
| 摄像头手势识别 | 开/关 | 开 |
| 语音识别 | 开/关 | 开 |
| 手势交互 | 开/关 | 开 |

### 8.3 手势教程

| 手势 | 动作 | 功能 |
|-----|------|------|
| ✊ 握拳 | 握紧拳头 | 暂停 |
| 🖐️ 张开手掌 | 伸开手掌 | 播放/继续 |
| 👆 食指指向 | 伸出手指 | 选择目标 |
| 👋 挥手 | 左右挥动 | 切换场景 |

### 8.4 推荐配置

**适合1.5-2岁宝宝：**
- 摄像头手势识别：关（手势太难）
- 语音识别：开 ✅
- 手势交互：关

**适合2-3岁宝宝：**
- 摄像头手势识别：开 ✅
- 语音识别：开 ✅
- 手势交互：开 ✅
