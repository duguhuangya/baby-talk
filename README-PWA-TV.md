# 咿呀学语 - PWA 电视端部署指南

本指南将帮助您将「咿呀学语」应用部署到安卓智能电视，让宝宝可以在客厅大屏幕上享受互动早教体验。

## 一、项目概述

「咿呀学语」是一款专为 1.5-3 岁宝宝设计的 AI 互动早教应用，支持手势识别和语音交互功能。通过本次 PWA 改造，应用现已支持在智能电视上独立运行，无需依赖手机或电脑。

### 核心功能

- **动物乐园**：6 种可爱动物，支持手势指向选择
- **家庭成员**：6 位家庭成员，支持语音识别匹配
- **手势控制**：握拳、张开手掌、指向、挥手四种手势
- **语音交互**：中文语音识别与合成播报
- **TV 遥控器导航**：完整的电视遥控器支持

## 二、技术实现

### 2.1 PWA 配置

项目使用 `vite-plugin-pwa` 插件实现渐进式 Web 应用功能，包括：

- **Service Worker**：实现应用离线缓存
- **Web App Manifest**：支持"安装到桌面"功能
- **全屏模式**：横屏沉浸式体验
- **缓存策略**：Google 字体、CDN 资源自动缓存

主要配置文件位于 `vite.config.ts`：

```typescript
VitePWA({
  registerType: 'autoUpdate',
  manifest: {
    name: '咿呀学语 - AI互动早教',
    short_name: '咿呀学语',
    display: 'fullscreen',
    orientation: 'landscape',  // 横屏模式
    theme_color: '#FF6B6B',
    background_color: '#FFF9F0',
    icons: [
      { src: '/icons/icon-192.png', sizes: '192x192' },
      { src: '/icons/icon-512.png', sizes: '512x512' }
    ]
  },
  workbox: {
    // 运行时缓存配置
  }
})
```

### 2.2 TV 端适配

#### 响应式布局

项目实现了完整的 TV 端响应式适配，主要断点如下：

| 设备类型 | 屏幕宽度 | 字号放大倍数 | 卡片尺寸 |
|---------|---------|-------------|---------|
| 手机 | < 480px | 1x | 120px |
| 平板 | 480-1024px | 1.2x | 160px |
| 电脑 | 1024-1280px | 1.4x | 200px |
| 普通电视 | 1280-1920px | 1.8x | 280px |
| 4K 电视 | ≥ 1920px | 2.2x | 320px |

TV 端专用样式位于 `src/styles/tv.css`，自动根据屏幕尺寸应用相应样式。

#### 遥控器导航

实现了完整的电视遥控器支持，宝宝可以通过遥控器自由操作：

| 按键 | 功能 | 说明 |
|-----|------|------|
| ↑ ↓ ← → | 移动焦点 | 在 3x2 网格中导航 |
| 确认键 | 选择 | 触发卡片点击效果 |
| 返回键 | 返回/家长模式 | 连续按 5 次打开设置 |
| 播放键 | 切换场景 | 在动物乐园和家庭成员间切换 |

核心导航逻辑位于 `src/composables/useTVNavigation.ts`，自动检测是否为 TV 设备并启用遥控器模式。

### 2.3 图标资源

项目生成了完整的 PWA 图标集，位于 `public/icons/` 目录：

- `icon-192.svg/png`：小尺寸图标（192x192）
- `icon-512.svg/png`：大尺寸图标（512x512）
- `icon-maskable.png`：支持安全区域的遮罩图标

## 三、快速部署

### 3.1 本地开发模式

首先克隆项目并安装依赖：

```bash
cd /workspace
npm install
npm run dev
```

服务启动后访问 `http://localhost:3000` 即可在浏览器中预览。

### 3.2 构建生产版本

```bash
npm run build
```

构建产物位于 `dist/` 目录，可直接部署到任意静态服务器。

### 3.3 局域网访问

开发模式下，项目已配置允许局域网访问。获取本机 IP 地址：

```bash
# Windows
ipconfig

# macOS / Linux
ifconfig
```

在电视浏览器中访问 `http://<电脑IP>:3000` 即可。

### 3.4 部署到云平台

#### Vercel 部署（推荐）

1. 安装 Vercel CLI：`npm i -g vercel`
2. 在项目根目录执行：`vercel`
3. 按照提示完成部署
4. 获取生成的访问地址

#### Netlify 部署

1. 将 `dist` 目录拖拽到 Netlify 拖放区域
2. 自动生成访问地址

#### 阿里云 OSS

1. 创建 OSS Bucket，开启静态网站托管
2. 上传 `dist` 目录所有文件
3. 配置自定义域名（可选）

## 四、电视端使用指南

### 4.1 打开应用

1. 确保电视和电脑在同一局域网内
2. 在电视浏览器中访问部署地址
3. 等待应用加载完成

### 4.2 安装到电视桌面

大多数安卓电视支持将 PWA"安装"到桌面：

1. 在浏览器中打开应用
2. 找到浏览器菜单中的"添加到主屏幕"或"安装"选项
3. 确认安装后，应用图标会出现在电视桌面

### 4.3 操作方式

#### 遥控器操作

- 使用方向键移动焦点，被选中的卡片会高亮显示
- 按确认键选择当前卡片
- 按返回键连续 5 次可打开家长设置面板

#### 手势操作（如已启用摄像头）

- 张开手掌：播放/继续
- 握拳：暂停
- 指向：选择目标
- 挥手：切换场景

#### 语音操作（如已启用语音识别）

- 在家庭成员场景说出称谓（如"妈妈"、"爷爷"）
- 应用会自动匹配并展示

### 4.4 家长设置

连续按 5 次遥控器返回键，可打开家长设置面板：

- 开启/关闭摄像头手势识别
- 开启/关闭语音识别功能
- 开启/关闭手势交互功能
- 查看手势操作教程

## 五、离线使用

部署后的应用支持完全离线使用：

1. 首次访问时，Service Worker 会缓存所有必要资源
2. 后续访问即使没有网络也能正常运行
3. 离线状态下语音识别功能不可用（需网络）

## 六、已知限制

1. **摄像头功能**：电视端使用摄像头需要外接 USB 摄像头，建议使用高清网络摄像头以获得最佳手势识别效果
2. **语音识别**：电视端麦克风拾音质量可能不如专业设备，建议在安静环境中使用
3. **浏览器兼容性**：部分老旧电视浏览器可能不完全支持 PWA，建议使用主流电视浏览器（如当贝浏览器、UC 浏览器 TV 版等）

## 七、故障排除

### 应用无法加载

- 检查网络连接
- 确认电脑防火墙允许 3000 端口
- 尝试刷新页面或清除浏览器缓存

### 遥控器无响应

- 确保电视浏览器获得焦点
- 尝试按电视遥控器的"确定"或"OK"键激活浏览器
- 检查浏览器设置中是否禁用了键盘导航

### 手势识别不准确

- 确保光线充足
- 摄像头对准宝宝，手部动作在摄像头视野内
- 尝试调整摄像头位置或角度

## 八、开发建议

### 生产环境优化

1. 使用 HTTPS 部署（部分浏览器功能需要安全上下文）
2. 配置 CDN 加速资源分发
3. 考虑使用 WebRTC 进行远程摄像头控制

### 后续功能扩展

1. 添加更多动物和场景
2. 支持多人同时识别
3. 增加游戏化元素（如积分系统）
4. 支持家长手机端远程控制
